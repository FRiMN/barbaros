# Barbaros

[English Documentation](../README.md)

**Приложение для перевода на основе ИИ на рабочем столе**

Barbaros — это легковесное приложение для рабочего стола, которое предоставляет мгновенные переводы на основе ИИ через системный трей. Просто скопируйте текст в буфер обмена и получите быстрые переводы с помощью горячей клавиши.
> **Примечание:** Тестировалось только на Linux, но может также работать на Windows и macOS.

## Особенности

- **🚀 Интеграция с системным треем**: Работает тихо в фоновом режиме, не загружая рабочий стол
- **📋 Перевод из буфера обмена**: Автоматически переводит текст из буфера обмена
- **🤖 На основе ИИ**: Использует продвинутые модели ИИ через несколько поставщиков LLM (Ollama, OpenAI, Anthropic и более чем 40 других) для точных переводов
- **🔒 Приоритет приватности**: Обработка переводов локально с Ollama или безопасно с облачными поставщиками (защита ваших данных зависит от выбранного поставщика)
- **🏠 Возможность работы без интернета**: Работает полностью офлайн с локальными поставщиками (например, Ollama), как только модели будут загружены
- **⚡ Быстрый доступ**: Мгновенное появление окна с настраиваемыми горячими клавишами
- **🔄 Различные методы связи**: Использует DBus и сигналы Unix для надежной работы

## Предварительные требования

Barbaros использует SDK [any_llm](https://github.com/mozilla-ai/any-llm) для поддержки более чем 40 разных поставщиков LLM. Вы можете выбрать работу с локальными поставщиками, такими как Ollama, или облачными поставщиками, такими как OpenAI, Anthropic и другими.

### Локальный поставщик: Ollama (по умолчанию)

Barbaros поставляется с Ollama, заранее настроенным как поставщик по умолчанию. Для локальных, приватно-ориентированных переводов установите [Ollama](https://ollama.ai/):

**Установка на Linux:**
```bash
curl -fsSL https://ollama.com/install.sh | sh
```

**Загрузите модель для перевода:**
После установки Ollama загрузите модель для перевода:
```bash
ollama pull llama3.1
```

Просмотрите доступные модели в [официальном каталоге Ollama](https://ollama.com/search).

### Облачные поставщики

Для облачных поставщиков вам потребуется:
- **API-ключ**: Получите API-ключ с веб-сайта выбранного поставщика
- **Доступ к модели**: Убедитесь, что у вас есть доступ к нужным моделям

Популярные облачные поставщики включают:
- **OpenAI**: Требуется [API-ключ](https://platform.openai.com/api-keys)
- **Anthropic**: Требуется [API-ключ](https://console.anthropic.com/)
- **Google Gemini**: Требуется [API-ключ](https://ai.google.dev/)
- **И более чем 40 других**, поддерживаемых any_llm

> **Примечание:** Облачные поставщики отправляют ваш текст на их серверы. Перед использованием ознакомьтесь с их политиками приватности.

## Установка

### Вариант 1: Пакет Flatpak (рекомендуется)

**Скачайте и установите:**
1. Получите последний файл `.flatpak` с [GitHub Releases](https://github.com/FRiMN/barbaros/releases)
2. Установите пакет:
   ```bash
   cd ~/Downloads
   flatpak install Barbaros-v1.2.3.flatpak
   ```

**Запуск:**
- **Терминал**: `flatpak run io.github.frimn.barbaros` или `flatpak run io.github.frimn.barbaros --opened` для открытия главного окна при запуске
- **GUI**: Найдите "Barbaros" в меню приложений

**Быстрый перевод:**
```bash
flatpak run io.github.frimn.barbaros --popup
```
*Совет: Свяжите эту команду с горячей клавишей для мгновенного доступа*

> **Примечание:** Ручная установка Flatpak не поддерживает автоматические обновления. Проверяйте GitHub-релизы для новых версий.

### Вариант 2: Установка для разработки

**Клонирование и настройка:**
```bash
git clone https://github.com/FRiMN/barbaros.git
cd barbaros
uv sync
```

**Запуск приложения:**
```bash
# Запуск приложения в системной строке
uv run barbaros

# Быстрый поп-ап для перевода
uv run barbaros --popup

# Запуск приложения с открытым главным окном без перевода
uv run barbaros --opened
```

## Использование

1. **Запустите приложение**: Запустите Barbaros, он появится в системном трее
2. **Выберите поставщика и модель** (опционально):
   - Откройте окно приложения
   - Во вкладке "Текст" или "OCR" выберите поставщика и модель из раскрывающегося списка
   - По умолчанию: Ollama с вашей первой доступной моделью
3. **Настройте поставщиков** (опционально):
   - Перейдите на вкладку "Настройки"
   - Нажмите "Управление поставщиками", чтобы добавить/отредактировать/удалить поставщиков
   - Нажмите кнопку обновления рядом с поставщиком, чтобы загрузить доступные модели
4. **Переведите текст**:
   - Скопируйте любой текст, который хотите перевести, в буфер обмена
   - Используйте команду `--popup`, или щелкните значок в системном трее
   - В окне перевода выберите целевой язык
   - Нажмите кнопку "Перевести" или используйте сочетание клавиш `Ctrl+Return`
5. **Получите результаты**: Окно перевода появится с вашим переведенным текстом

## Поставщики

Barbaros поддерживает более чем 40 поставщиков LLM через SDK [any_llm](https://github.com/mozilla-ai/any-llm), предоставляя вам гибкость выбора между локальными и облачными моделями:

### Поставщик по умолчанию
- **Ollama**: Преднастроен для локальных, приватно-ориентированных переводов

### Поддерживаемые поставщики
Приложение поддерживает поставщиков, включая:
- **Локальные**: Ollama, Llama.cpp, Llamafile, vLLM, LM Studio
- **Основные облачные**: OpenAI, Anthropic, Google Gemini, Azure
- **Специализированные**: Groq, Together AI, Cerebras, Fireworks, Sambanova
- **Корпоративные**: AWS Bedrock, IBM watsonx, Databricks, Vertex AI
- **И более чем 30 других**: См. [документацию any_llm](https://github.com/mozilla-ai/any-llm) для полного списка

### Управление поставщиками

1. **Откройте настройки**: Перейдите на вкладку "Настройки" в Barbaros
2. **Управление поставщиками**: Нажмите кнопку "Управление поставщиками"
3. **Добавить поставщика**:
   - Нажмите "Добавить поставщика"
   - Введите имя (например, "My Ollama")
   - Выберите тип поставщика из раскрывающегося списка (OpenAI, Anthropic и т.д.)
   - Введите API-ключ (требуется для облачных поставщиков)
   - Введите URL API, если используете измененный базовый URL к API (опционально)
4. **Загрузите модели**: Нажмите кнопку обновления рядом с поставщиком, чтобы получить доступные модели
5. **Выберите модель**: Выберите из раскрывающегося списка поставщика/модели во вкладках "Текст" или "OCR"

### Примеры настройки поставщиков

**OpenAI:**
- Тип: `openai`
- API-ключ: Ваш API-ключ OpenAI с https://platform.openai.com/api-keys
- URL API: Оставьте пустым (использует значение по умолчанию)

**Anthropic:**
- Тип: `anthropic`
- API-ключ: Ваш API-ключ Anthropic с https://console.anthropic.com/
- URL API: Оставьте пустым (использует значение по умолчанию)

**Кастомный OpenAI:**
- Тип: `openai`
- API-ключ: API-ключ вашего провайдера
- URL API: `https://your-endpoint.com/v1`

**Ollama (локальный):**
- Тип: `ollama`
- API-ключ: Оставьте пустым
- URL API: Оставьте пустым (по умолчанию используется `http://localhost:11434`)

### Выбор модели

Каждый поставщик может иметь несколько моделей. Выберите нужную модель из раскрывающегося списка:
- Названия поставщиков отображаются как верхние элементы
- Модели появляются как дочерние элементы под каждым поставщиком
- Выбор сохраняется между сессиями

## Как это работает

Barbaros использует двойную систему коммуникации для надежной функции поп-апа:

- **Основной**: Сообщения DBus для бесшовной интеграции с рабочим столом
- **Резервный**: Сигналы Unix (`SIGUSR1`) для надежной работы

Когда вы используете аргумент `--popup`, команда связывается с запущенным экземпляром приложения, который затем:
1. Приводит главное окно к переднему плану
2. Получает текст из буфера обмена
3. Обрабатывает перевод через выбранный поставщик и модель
4. Отображает результаты

**Коммуникация с поставщиками:**
- **Локальные поставщики** (например, Ollama): Переводы обрабатываются офлайн на вашем компьютере
- **Облачные поставщики** (например, OpenAI, Anthropic): Безопасные вызовы API к серверам поставщика
- **Двухканальная система коммуникации**: Сообщения DBus + сигналы Unix для надежной функции поп-апа

## Отказ от ответственности за качество перевода

**Важное уведомление:** Качество перевода полностью зависит от выбранной вами модели ИИ и ее возможностей. Barbaros — это инструмент, который упрощает процесс перевода, но не гарантирует точность, полноту или соответствие переводов.

- Точность перевода варьируется между разными парами языков
- Сложные технические, юридические или медицинские тексты могут требовать профессионального перевода человеком
- Всегда проверяйте и уточняйте переводы для важных документов
- Приложение не несет ответственности за последствия, вызванные ошибками перевода

Для критически-важных переводов рекомендуется обращаться к профессиональным услугам перевода.
